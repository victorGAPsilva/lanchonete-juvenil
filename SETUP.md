# Setup Instructions - Lanchonete Juvenil

## Local Development Setup

### Option 1: Using SQLite (Recommended for Quick Start)

SQLite é pré-configurado e pronto para usar. Apenas execute:

```bash
php artisan migrate
php artisan db:seed
php artisan serve
```

O site estará disponível em `http://localhost:8000`

### Option 2: Using MySQL

#### Prerequisites:
- MySQL 8.0+ instalado e rodando
- Credenciais: root/password ou qualquer usuário com permissão para criar banco

#### Steps:

1. **Create Database:**
```bash
mysql -u root -p
mysql> CREATE DATABASE lanchonete_juvenil CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
mysql> EXIT;
```

2. **Configure .env:**
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=lanchonete_juvenil
DB_USERNAME=root
DB_PASSWORD=your_password
```

3. **Run Migrations:**
```bash
php artisan migrate
php artisan db:seed
php artisan serve
```

#### Alternative: Using Docker

```bash
docker run --name mysql-lanchonete \
  -e MYSQL_ROOT_PASSWORD=root \
  -e MYSQL_DATABASE=lanchonete_juvenil \
  -p 3306:3306 \
  -d mysql:8.0

# Then run:
php artisan migrate
php artisan db:seed
php artisan serve
```

### Option 3: Using Docker Compose

Create `docker-compose.yml`:

```yaml
version: '3.8'

services:
  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: root
      MYSQL_DATABASE: lanchonete_juvenil
    ports:
      - "3306:3306"
    volumes:
      - mysql_data:/var/lib/mysql

  app:
    build: .
    ports:
      - "8000:8000"
    depends_on:
      - mysql
    environment:
      - DB_HOST=mysql

volumes:
  mysql_data:
```

Then:
```bash
docker-compose up -d
php artisan migrate
php artisan db:seed
```

## Development

### Running the Dev Server
```bash
php artisan serve
# Access at http://localhost:8000
```

### Database Commands
```bash
# Run migrations
php artisan migrate

# Seed database with dummy data
php artisan db:seed

# Rollback migrations
php artisan migrate:rollback

# Reset database
php artisan migrate:fresh --seed
```

### Creating Models/Controllers/Migrations
```bash
# Create model with migration
php artisan make:model ModelName -m

# Create controller
php artisan make:controller ControllerName

# Create middleware
php artisan make:middleware MiddlewareName
```

## Environment Variables Required

### Application
- APP_NAME=LanchoneteJuvenil
- APP_ENV=production
- APP_KEY=(generated by php artisan key:generate)
- APP_DEBUG=false
- APP_URL=https://yourdomain.com

### Database
- DB_CONNECTION=mysql (or sqlite)
- DB_HOST=your_host
- DB_DATABASE=lanchonete_juvenil
- DB_USERNAME=your_user
- DB_PASSWORD=your_password

### Payments (Mercado Pago)
- MP_PUBLIC_KEY=your_public_key
- MP_ACCESS_TOKEN=your_access_token

### Messaging (WhatsApp)
- WHATSAPP_PHONE=5511999999999
- WHATSAPP_TOKEN=your_token

## Troubleshooting

**Migration Errors:**
```bash
# Clear composer cache
composer dump-autoload

# Refresh database
php artisan migrate:fresh

# Check migration status
php artisan migrate:status
```

**Database Connection Issues:**
```bash
# Test connection
php artisan tinker
>>> DB::connection()->getPdo()
```
